# desmos-equations
## All desmos equations and the coded app that takes edge cases to form Catmull-Rom spline segment, and using algorithms to form cubic Bézier curves.

How to use:
Save the desmos_tracer_updated.html file and open it. Insert your image and adjust settings to fine tune the resultant equations.
